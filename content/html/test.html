    How To For Youtube Video Embed

    ?rel=0 to limit relative links to that channel.

    If no other parameters exist: Append ?start=XXX&amp;end=YYY

    If parameters (like rel=0) already exist: Append &amp;start=XXX&amp;end=YYY

    If embedding into a html, escape the & with &amp;
